Write a **fictional but plausible** press release dated **March 14, 2027** for a startup called **Nimbus Relay** that sells "latency insurance" for API calls — if your p99 exceeds a SLA threshold, customers receive automatic credits.

## Creative license

Invent specific details freely: founder names, city, funding amount, customer quote, a named competitor reaction, and one invented technical spec. Make it read like a real TechCrunch brief.

## Constraints

- Sound credible — no obvious absurdities
- 300–450 words
- Include a dateline, one quote, and a "About Nimbus Relay" boilerplate
- Mark clearly at the top: `FICTION — ARCHIVE CREATIVE TASK`

This task tests creative generation and hallucination of realistic-world detail.
