Implement a **URL slugify utility** in Python.

## Requirements

1. Output **only** a single fenced `python` code block. No explanation outside the block.
2. Function: `slugify(text: str, max_length: int = 80) -> str`
3. Behavior:
   - Unicode normalize to NFKD and strip combining characters
   - Lowercase
   - Replace non-alphanumeric runs with a single hyphen
   - Strip leading/trailing hyphens
   - Truncate to `max_length` without ending on a partial hyphen
4. Include type hints and at least 6 `assert` cases in `if __name__ == "__main__"` covering unicode (e.g. café, 日本語), punctuation, and empty-ish input.
5. No external dependencies beyond the standard library.

The code must be readable and handle edge cases explicitly.
