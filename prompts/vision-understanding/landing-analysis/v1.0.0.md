You are given a screenshot of a generated landing page for a developer tool called **Drift**.

Analyze the image and respond in plain text with these sections:

## Layout map
Describe the vertical section order (navigation, hero, features, etc.).

## Visual hierarchy
What draws the eye first? Note typography scale, contrast, and spacing.

## Content inventory
List every major headline, CTA label, and navigation item visible.

## Accessibility observations
Note contrast, tap target sizing, and any obvious issues.

## One-sentence summary
What is this page trying to communicate?

Be specific to what you see in the image. Do not invent sections that are not visible.
