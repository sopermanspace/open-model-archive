Create a complete, production-quality single-file HTML page for a fictional indie game called **Starfield Salvage**.

## Product

A cozy co-op space salvage game for two players. Tagline: "Repair ships. Argue about wiring. Save the station."

## Requirements

1. Output **only** a single fenced `html` code block. No explanation outside the code block.
2. Self-contained HTML with embedded CSS only. Google Fonts allowed (choose something playful — not Inter).
3. **Light theme** with warm pastels or soft cosmic colors — must feel distinctly different from a dark developer-tool landing page.
4. Sections: navigation, hero with email waitlist form, three feature cards with emoji or simple CSS shapes, a "meet the crew" duo section, footer.
5. No gradients, glassmorphism, or heavy animation.
6. Semantic HTML5. Accessible contrast. Responsive layout.
7. Primary CTA button text: "Join the waitlist".
8. Fictional copy only — no real companies or games.

The page should feel like a charming indie game marketing site, not enterprise SaaS.
