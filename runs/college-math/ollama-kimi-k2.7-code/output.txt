1. **Calculus — Derivative of f(x) = x³ · ln(x)**

Use the product rule:  
(u·v)′ = u′·v + u·v′, where u = x³ and v = ln(x).

u′ = 3x²  
v′ = 1/x

So:

f′(x) = 3x²·ln(x) + x³·(1/x)  
       = 3x²·ln(x) + x²  
       = x²(3·ln(x) + 1)

**Final answer:** f′(x) = x²(3 ln x + 1)

2. **Calculus — Evaluate ∫₀¹ (2x + 1) dx**

Find the antiderivative:

∫ (2x + 1) dx = x² + x

Evaluate from 0 to 1:

[x² + x]₀¹ = (1² + 1) − (0² + 0) = 2 − 0 = 2

**Final answer:** 2

3. **Linear algebra — Inverse of A = [[2, 1], [1, 3]]**

For a 2×2 matrix A = [[a, b], [c, d]],

A⁻¹ = (1/det(A)) · [[d, −b], [−c, a]]

det(A) = (2)(3) − (1)(1) = 6 − 1 = 5 ≠ 0, so the inverse exists.

A⁻¹ = (1/5) · [[3, −1], [−1, 2]]

        = [[3/5, −1/5],
           [−1/5, 2/5]]

**Final answer:** A⁻¹ = [[3/5, −1/5], [−1/5, 2/5]]

4. **Linear algebra — Solve the system**

2x + y = 7  
x − 3y = −1

Solve the second equation for x:

x = 3y − 1

Substitute into the first equation:

2(3y − 1) + y = 7  
6y − 2 + y = 7  
7y = 9  
y = 9/7

Now find x:

x = 3(9/7) − 1 = 27/7 − 7/7 = 20/7

**Final answer:** x = 20/7, y = 9/7

5. **Probability — Exactly 3 heads in 5 fair coin flips**

This is a binomial probability:

P(X = k) = C(n, k) · p^k · (1−p)^(n−k)

Here n = 5, k = 3, p = 1/2.

C(5, 3) = 5! / (3!·2!) = 10

P = 10 · (1/2)³ · (1/2)²  
  = 10 · (1/8) · (1/4)  
  = 10/32  
  = 5/16  
  = 0.3125

**Final answer:** 5/16 (or 0.3125)
