Editorial Restaurant Site

Warm typography-led restaurant page — editorial layout vs product marketing.

Prompt v1.0.0

Create a complete, production-quality single-file HTML page for a fictional restaurant called **Ember & Oak**.

## Concept

Modern fine dining in Portland. Tagline: "Wood fire. Local grain. Long tables."

## Requirements

1. Output **only** a single fenced `html` code block. No explanation outside the code block.
2. Self-contained HTML with embedded CSS only. Use an elegant serif + clean sans pairing via Google Fonts.
3. **Warm editorial aesthetic** — cream or off-white background, restrained earth tones, generous whitespace. Must not look like a SaaS product page.
4. Sections: minimal nav, hero with reservation CTA, chef's story (short prose), sample menu (3 courses, no prices), hours & location, footer.
5. Typography-led design — large pull quotes, subtle borders, no card grids unless minimal.
6. Semantic HTML5. Accessible contrast. Responsive layout.
7. Reservation button text: "Book a table".
8. Fictional copy only.

The page should feel like a magazine-quality restaurant feature, not a startup landing page.
