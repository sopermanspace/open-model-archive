App Logo Mark

Filled geometric app icon — compare against stroke-based weather icons.

Prompt v1.0.0
ID: app-logo SHA-256: 7899aa77f216918a… 
Create a **single app logo mark** as inline SVG for a fictional productivity app called **Anchor**.

## Requirements

1. Output **only** a single fenced `svg` code block. No explanation outside the block.
2. One icon only — **not** an icon set. viewBox `0 0 48 48`.
3. Style: **filled geometric shapes** (not stroke-based line icons). 2–3 flat colors maximum.
4. Concept: suggest "anchor" or "stability" abstractly — no literal clipart anchor required.
5. No text, no gradients, no filters, no raster images.
6. Must look like a credible app icon — balanced, centered, recognizable at 32px.

No labels. No explanation outside the SVG code block.
