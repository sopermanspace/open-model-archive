About

Open Model Archive is a public, reproducible record of what AI models generate when given the same real-world tasks.

What this is

Not a leaderboard. Not a single score. A transparent archive where developers can inspect exact outputs, artifacts, timing, and token usage side by side.

How it works

  1. Versioned prompts live in the repository.
  2. The build pipeline executes each task against registered models.
  3. Outputs, metadata, and screenshots are committed as public artifacts.
  4. A static site is generated for effortless comparison.

Model providers

  • Ollama — open-source and cloud-backed models via the local Ollama API.
  • Provider CLI — frontier models via official CLI integrations where available.
  • Direct APIs — OpenAI, Anthropic, OpenRouter, Together, Fireworks, Sarvam.

Task categories

  • Website generation — full HTML pages with snapshot and live preview.
  • Code generation — extractable source artifacts.
  • Blog writing — long-form markdown posts.
  • SVG generation — inline vector graphics.
  • Web grounding — answer only from provided source excerpts.
  • Vision understanding — analyze reference images.
  • Design critique — expert designer review of visual outputs.
  • Humor & creative — tone, wit, and plausible fiction.
  • Math — elementary through college-level problems.
