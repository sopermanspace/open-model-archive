Features

Catch drift before it ships

Environment parity breaks silently. Drift makes the invisible visible — in your terminal, in CI, and in your pull requests.

Δ

Deep environment diffing

Compares env vars, config files, feature flags, and infrastructure state between any two targets. Supports .env, YAML, JSON, and Terraform backends out of the box.

CI

CI-native workflow

Add a single step to your pipeline. Drift exits non-zero on drift, blocks merges automatically, and posts a structured diff comment to your PR.

⊘

Allowlist & baselines

Not every difference is a bug. Define expected drift with allowlists, set baselines per environment pair, and ignore noisy keys with glob patterns.

Configuration

Set up in minutes

Drop a .drift.yml in your repo root. Drift discovers your environments from your existing tooling — no migration required.

  • Works with Kubernetes secrets, Vault, and flat files
  • Supports GitHub Actions, GitLab CI, and CircleCI
  • Outputs SARIF for security dashboard integration
