Know before you deploy.

A fast, minimal CLI that detects configuration drift between your staging and production environments before things break.

Start drift scan
$ drift scan --source staging --target prod Scanning 412 configuration files... ✖ Drift detected in config/database.yml staging: pool_size: 20 prod: pool_size: 5 Run `drift sync` to resolve or `drift ignore` to skip.

Universal Compatibility

Works out of the box with Kubernetes manifests, Terraform state files, Docker Compose, and raw .env configurations.

Surgical Precision

Drift doesn't just diff text. It parses your YAML, JSON, and TOML to understand structural changes and semantic meaning.

CI/CD Native

Drop a single binary into GitHub Actions, GitLab CI, or CircleCI. Fail builds automatically when unexpected drift occurs.
